TERMOS ESTRANGEIROS DE TI

Termo Extenso Tradução Função/significado
if   se Comando utilizado em Programação para montar blocos de código que podem ou não ser executados de acordo com a avaliação de uma expressão lógica.
else   se não comando de programação usado como condição contraria ao If
hd   hard disk disco rígido Componente usado em computadores eletrônicos para armazenar dados de maneira permanente.
SSD   Solid State Drive unidade de estado sólido É um tipo de dispositivo de armazenamento de dados que usa memória flash para armazenar informações, ao contrário dos discos rígidos
for   para Comando utilizado em Programação para montar blocos de código em condição de repetição caso o usuário já conheça a condição de parada
Do while   faça enquanto Comando utilizado em Programação para montar blocos de código em condição de repetição, executando o codigo uma vez antes da condição de parada ser aceita.
While   enquanto Comando utilizado em Programação para montar blocos de código em condição de repetição, executando o codigo apenas quando a condição de parada não for verdadeira.
Memória RAM   Random Access Memory Memória de Acesso Randômico Memória principal do computador, responsável pelo armazenamento dos programas que estão em execução no momento do uso, e que é apagada quando o compútador é desligado.
Memória ROM   READ ONLY MEMORY MEMÓRIA SOMENTE LEITURA É um tipo de memória não volátil usada em computadores e outros dispositivos eletrônicos para armazenar permanentemente dados que não precisam ser alterados.
SOFTWARE   Programas/Aplicativos Parte lógica do sistema, integrando seus aplicativos e programas, ou seja, a parte "não física" do computador.
hardware   Equipamentos/Perifericos Parte fisíca do computador, como seus componentes, dispositivos de entrada e saida, o que pode ser tocado.
File Explorer   Explorador de Arquivos Gerenciador de arquivos que cria, apaga, copia, renomeia e organiza arquivos e pastas em diferentes mídias.
System Call   Chamadas de Sistema É o processo de chamadas pelo qual um programa de usuário solicita serviços ao núcleo (kernel) do sistema operacional
CLOUD COMPUTING   computação em nuvem permite que os dados e os aplicativos sejam armazenados em servidores remotos, acessíveis pela internet, em vez de no dispositivo local. Isso oferece flexibilidade, escalabilidade e redução de custos.
linker   ligador é o utilitário responsável por gerar, a partir de um ou mais módulos objeto, um único programa executável. Resolver todas as referências simbólicas existentes entre os módulos - objeto, reservar memória para a execução do programa e determinar uma região da memória onde o programa será carregado para sua execução.
loader   carregador é o utilitário responsável por colocar fisicamente na memória principal um programa para sua execução. Pode permitir que um programa seja carregado em regiões diferentes toda vez que for trazido para a memória
WWW   world wide web rede com amplitude mundial é um sistema de informações interligadas, onde documentos e outros recursos são acessados através da internet, utilizando URLs e hiperlinks.
Debugging   Depuração É o processo de encontrar e corrigir erros em um programa. Ele envolve a análise do código e a identificação do problema.
Compile   Compilador É um programa que converte o código fonte em um código executável. Ele verifica a sintaxe do código e o transforma em uma linguagem que a máquina possa entender.
browser   Navegador Conhecido como navegador da web, é um software que permite aos usuários acessar e visualizar páginas na internet. Ele atua como uma interface entre o usuário e a World Wide Web, interpretando códigos como HTML, CSS e JavaScript para exibir conteúdos de forma visual e interativa.
Blockchain   Blockchain É uma tecnologia de registro distribuído que garante a segurança e a transparência das transações. É a base para as criptomoedas, como o Bitcoin e o Ethereum, e também é usado em outras áreas, como contratos inteligentes e cadeias de suprimento.
IoT   internet of things Internet das coisas envolve a interconexão de dispositivos físicos (como eletrodomésticos, carros, sensores e equipamentos industriais) à internet, permitindo a coleta e troca de dados entre eles de forma inteligente.
AI   Artificial Intelligence Inteligência artificial refere-se à criação de sistemas capazes de realizar tarefas que normalmente exigiriam inteligência humana, como reconhecimento de voz, visão computacional, tomada de decisões e aprendizado a partir de dados.
SO   operating system Sistemas Operacionais É um software que gerencia os recursos de um dispositivo, como a memória e o processamento, e permite a execução de programas e aplicativos.
CPU   central processing unit Unidade Central de Processamento É o componente principal de um computador responsável por executar instruções e realizar cálculos. Ela atua como o "cérebro" do sistema, gerenciando e processando todas as informações necessárias para o funcionamento do computador.
BIOS   Basic Input/Output System Sistema básico de Entrada/Saída Inicializa o hardware e inicia o carregamento do sistema operacional ao ligar o computador.
POST   power-on self-test Autoteste ao Ligar Realiza, no software, um autoteste para verificar se os componentes essenciais do hardware estão funcionando corretamente.
AP   Access Point Ponto de acesso é um dispositivo de rede que permite que dispositivos como laptops, smartphones e tablets se conectem a uma rede wi-fi.
NIC   network card Placa de rede É um hardware de computador que permite que um computador se conecte a uma rede.
Cable Cross-Over   Cabo de cruzamento É o cabo que faz o cruzamento externo, ou seja, interliga a saída de dados do primeiro micro à entrada de dados do segundo micro e vice-versa.
MODEM   modulador-demodulador É um dispositivo que converte sinais digitais em sinais analógicos e vice-versa.
LAN   Local Área Network Rede Local São redes de computadores locais que ocupam um pequeno espaço físico.
MAN   Metropolitan Area Network Rede Metropolitana Uma rede de computadores metropolitana, que pode abranger um grupo de escritórios vizinhos ou uma cidade inteira e pode ser privada ou pública.
WAN   Wide Area Network Rede geograficamente distribuída Abrange uma ampla área geográfica, isto é uma rede que alcança grandes p roporções, ou seja, uma rede onde a distância pode variar, de cidades, de países ou até mesmo continentes diferentes.
STP   Shielded Twisted Pair Par Trançado com Blindagem Individual O cabo STP possui uma blindagem metálica em torno de cada par de fios torcidos.
F/FTP   Foiled Twisted Pair Blindagem com folha O cabo F/FTP possui uma folha de metal (geralmente de alumínio) envolvendo cada par de fios torcidos e uma folha envolvendo todos pares de fios.
S / FTP   Screened Foiled Twisted Pair Par Trançado com Blindagem Individual e Global Este tipo de cabo combina a blindagem individual em cada par de fios torcidos com folha e uma blindagem global em torno de todos os pares com malha metálica.
Package   Pacote Estrutura organizacional para classes, em programação.
DMA   Direct Memory Access Acesso Direto à Memória O controlador de E/S transfere dados diretamente entre o dispositivo e a memória, sem a intervenção do processador durante a transferência, só interrompendo o processador ao final da operação.
bit rate   Taxa de transmissão É a velocidade na qual os dados são transferidos de um ponto para outro em um período de tempo. Medida em bits por segundo
Patch Cord   cordões de ligação são cabos curtos (0,5 a 5m) já montados com proteção no conector.
SMF   Single Mode Fiber Fibra Óptica Monomodo Cabo de fibra óptica em que a fibra tem núcleo menor e transmite apenas um sinal por vez
MMF   Multi Mode Fiber Fibra Óptica Multimodo Cabo de fibra óptica em que a fibra tem núcleo maior e transmite vários sinais por vez mas atinge menores distâncias.
PC fiber   Physical Contact conector com contato físico A superfície de contato do conector (PC) é polida no formato convexo.
UPC fiber   Ultra Physical Contact conector com contato ultra físico São polidos sem ângulo muito parecido com o PC, mas eles possuem uma ligeira curvatura para melhor alinhamento do núcleo.
